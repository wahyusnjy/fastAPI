# Aplikasi Sample Bootcamp dengan Fast API #


## Setup Project ##

1. Clone Repository

2. Create venv

    ```sh
    python3 -m venv .venv
    ```

3. Activate virtual environment

    ```sh
    source .venv/bin/activate
    ```

4. Install dependencies

    ```
    pip install -r requirements.txt
    ```

5. Run test

    ```
    pytest
    ```

6. Run App

    ```
    fastapi dev app/main.py
    ```
